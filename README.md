# 🚀 SmartApply.AI

**SmartApply.AI** is a full-stack, AI-powered job application assistant that helps users tailor resumes, generate cover letters, practice interviews, and track job applications — all from one intuitive dashboard.

---

## 🌟 Overview

SmartApply.AI simplifies the entire job application process using AI. From optimizing resumes to preparing for interviews, it acts as your personal career assistant.

---

## ✨ Features

### 🧾 AI Resume Optimizer
- Upload resume (PDF/DOCX)
- Paste job description
- Generate:
  - Tailored resume
  - Cover letter
  - Match score (ATS-friendly)

---

### 🧠 AI Interview Practice
- Generate job-specific interview questions
- STAR-based answer suggestions
- Save & revisit past sessions

---

### 📊 Application Tracker
- Add and manage job applications
- Track status:
  - Applied
  - Interviewing
  - Rejected
  - Offer
- Add notes for each application

---

### 🌐 Portfolio Builder
- Auto-generate portfolio from resume
- Editable sections:
  - Projects
  - Skills
  - Experience
- Shareable public link

---

### 🔐 Authentication & Security
- Secure login/signup system
- JWT-based authentication
- Password hashing with bcrypt
- Protected routes

---

## 🛠️ Tech Stack

| Layer        | Technology                          |
|--------------|------------------------------------|
| Frontend     | React.js, TailwindCSS, TypeScript  |
| Backend      | Node.js, Express.js                |
| Database     | MongoDB                            |
| AI Engine    | Google Gemini API                  |
| File Parsing | pdfplumber, python-docx            |
| Auth         | JWT, bcrypt                        |
| Deployment   | Vercel (Frontend), Render (Backend)|
